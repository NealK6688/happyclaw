# 国内电子数据取证厂商和产品全景（2026-05）

> 读者：国内大型互联网公司数据安全 / 取证工程师
> 视角：企业自用 + 个人能力升级；公安/司法是国内厂商的主要客户，但本文不展开司法取证流程
> 范围：**PC / 服务器 / 桌面取证、磁盘镜像、内存取证、文件系统分析、artifact 收集**；移动取证产品仅在厂商同时做 PC 取证时附带提及
> 不在范围：实时 EDR / XDR、纯司法鉴定服务机构（如安证、存证云）、纯手机取证厂商（如奇虎、龙信偏移动模块）

---

## 1. 执行摘要

国内电子数据取证市场是一个**公安/司法采购拉动、商业市场弱**的细分赛道，整体规模约 30-50 亿人民币（远小于 EDR / 态势感知），却孕育了一家上市公司（国投智能/原美亚柏科 300188，2024 年营收 17.69 亿、同比下滑 10.82%）和一批专精特新厂商。市场格局可清晰分为三梯队：

- **Tier 1（取证主营，年营收过亿）**：**美亚柏科**（国内市占率 8 年第一、2023H1 公安市占率达 58%）、**奇安信盘古石**（2019 入局、2020 取证业务约 1 亿元、含手机取证）、**效率源**（成都，数据恢复起家、120+ 国家客户）。
- **Tier 2（取证 + 司法鉴定服务双轮）**：**弘连网络**（上海，70+ 专利、火眼手机取证 + 寻迹大数据分析）、**龙信科技**（苏州，公安部 2024 警用框采入围）、**四川金舟 / 金诺**（公开资料稀少，主要供四川/西南公安）。
- **Tier 3（取证不是主营、特定模块切入）**：**安天**（探海 NDR 带取证溯源、AVL 引擎可用于内存恶意代码识别，**没有正面取证一体机产品**）、**国保金泰**（主营安全隔离 / 单向导入，**取证不是主线**）、**齐力软件**（广州，公开材料极少，疑似已转型或低活跃）、**公安部三所**（认证机构 + 内部技术支持，**少量自研但不对外大规模销售**）。

**跟国外（EnCase / FTK / X-Ways / Magnet AXIOM）的关键差距**：
1. **底层格式解析深度** — 国内产品对 APFS / ReFS / 苹果生态 artifact、Windows 加密容器（BitLocker w/ TPM 2.0）的解析仍跟 X-Ways / AXIOM 有 1-2 年滞后；
2. **国际法庭认可度** — `取证大师` 在大陆外的法庭采信度低，跨境调查公司仍优选 EnCase；
3. **内存取证** — 国内厂商普遍把内存取证作为"功能集成项"而非旗舰产品，缺乏 Volatility / Rekall 这种社区驱动的插件生态；
4. **macOS 取证** — APFS 加密卷、Keychain、iCloud sync artifact 解析普遍弱于 BlackBag/Magnet；
5. **AI 辅助分析** — 国外 Magnet AXIOM Cyber、Cellebrite Pathfinder 的 LLM 集成更成熟，国内 Qiko 大模型一体机（2024 Q4 发布）刚起步。

**国内的反超点**：① **信创兼容**（统信 UOS / 麒麟 / 飞腾 / 龙芯 原生支持，国外产品几乎都没适配）；② **公安数据生态打通**（人口库 / 大情报平台 API、卡口车辆数据）；③ **国产应用 artifact 解析**（微信 / QQ / 抖音 / 拼多多 / 美团等 500+ 国产 App 数据库直读）；④ **驻场服务 + 渠道下沉**（市/县级公安都有售前售后）；⑤ **价格** — `取证大师` 专业版 3.5 万元/套（不含硬件） vs EnCase Forensic 一年订阅 4000+ 美元 + 培训。

**为什么是这 9 家**：覆盖了"取证主营 Tier 1 + 取证+司法混合 Tier 2 + 边缘玩家 Tier 3 + 监管端公安部研究所"四类角色。**有意省略**：纯手机取证厂商（如奇虎手机取证、北京网信易安 SIM 卡取证）、纯电子数据司法鉴定所（如安证、广东甄明，是消费方不是产品方）、阿里云/腾讯云的内部取证团队（不对外售）。

数据归因校准：公开市占率口径混乱，"58% 公安市占率"出自国投智能投资者关系材料 + 国盛证券 / 国泰君安研究报告（[国泰君安 美亚柏科深度](https://m.gelonghui.com/p/124433)、[国盛 2020 深度](https://finance.sina.cn/2020-05-29/detail-iirczymk4203725.d.html)），**不是 IDC 中国或赛迪顾问的原报告口径**，引用时请注意来源。

## 3. 厂商深度卡片

### 3.1 美亚柏科（国投智能 300188）

- **产品矩阵**：核心是 [取证大师](https://300188.cn/products/detail/40.html)（FM-2008P.V6 专业版 / 破冰版）+ 火眼系列（手机/数据库分析）+ 物联取证大师 + 取证云 SaaS 平台。硬件线 DC-4500 / DC-8200（高速镜像复制）、MagiCube（便携现勘）。2024 Q4 推出 [Qiko 大模型一体机](https://www.afzhan.com/news/detail/99990.html)，内置 70 亿参数公安大模型。
- **能力亮点**：① 国内首创 M01 镜像挂载技术，大文件复制 200MB/s；② [基于银河麒麟的全栈信创取证方案](https://kylinos.cn/solution/industry/shu-zi-zheng-fu-jie-jue-fang-an/36.html)；③ 国产应用 artifact 解析覆盖最全（500+ App）；④ 取证大师支持 BitLocker 解密、EFS 密钥提取、FAT32 恢复；⑤ 美亚杯大赛 10 余年沉淀的取证教学生态。
- **短板**：① 跨境法庭采信度低、英文文档稀少；② 内存取证不是旗舰、仍以集成功能存在；③ 公司 2023 业务下滑、人员减少 635 人（[CSReviews 财报解读](https://www.csreviews.cn/?p=8212)）；④ 央企背景（国投入主、更名国投智能）后客户结构进一步向公安/政企倾斜，纯商业客户接入度低。
- **客户类型**：公安、检察、法院、海关、纪委监委、央国企内审；互联网大厂极少。
- **价格**：取证大师专业版约 3.5 万元/套（不含硬件，[2018 公开渠道价](http://www.gzhaoan.com/product-45306-192285.html)），一体机 20-80 万元/台。中国政法大学采购 359 万元含舆情监测+大数据建模整套。
- **国外对标**：EnCase Forensic（软件分析） + Tableau TX1（硬件镜像）的国产化组合，但分析能力更接近 Magnet AXIOM Process。
- **CNVD / CNNVD 历史**：截至 2026-05，公开漏洞库（[cnvd.org.cn](https://www.cnvd.org.cn/)）未检索到取证大师 / 火眼的 CVE 条目，但取证软件本身处理大量untrusted disk image，仍存在 parser bug 风险（**未公开 ≠ 不存在**）。

### 3.2 奇安信盘古石

- **产品矩阵**：[盘古石取证](https://qz.qianxin.com/) 团队源自盘古实验室，主力产品 SafeAnalyzer（计算机取证分析）、SafeImager（现场取证）、SafeUnlocker（Windows 密码绕过）、[取证战星](https://www.qianxin.com/product/detail/pid/344) 综合一体机、手机取证分析系统、IoT 取证。
- **能力亮点**：① "漏洞思维"切入 — 盘古实验室原生的 0day 储备复用到取证 unlock；② [率先支持国产化信创环境](https://qz.qianxin.com/)；③ 现场快速采集 + 实验室深度分析两套并行；④ 取证战星硬件 + 软件深度耦合、模块化设计。
- **短板**：① 起步晚（2019），底层 artifact 库厚度不如美亚；② 跟奇安信主营 NDR / EDR / SOC 业务的销售协同有限、独立性强；③ 取证业务营收占奇安信整体 1% 以下，研发投入持续性需观察。
- **客户类型**：公安、央企内审；奇安信集团客户的 IR/取证延伸单。
- **价格**：未公开，类似一体机 30-60 万元区间。
- **国外对标**：偏向 Magnet AXIOM Cyber + Cellebrite Inseyets 的功能组合，但分析深度有差距。
- **CNVD 历史**：盘古石产品未检索到公开 CVE，但奇安信集团整体维护 [补天平台](https://qianxin.butian.net/) 向 CNVD 报送漏洞，公司侧漏洞响应流程成熟。

### 3.3 效率源（XLY Soft）

- **产品矩阵**：四川效率源信息安全技术股份有限公司（成都），数据恢复起家（[官网](https://www.xlysoft.net/)）。产品分三大类：① 数据恢复（FRS 领航者 X-AI 矩阵、171 项核心专利）；② 电子取证（AMF9300 极光手机多路快取、SPF 系列）；③ 现场勘验（[突发应急取证工具包](https://zhuanlan.zhihu.com/p/651789333)）。
- **能力亮点**：① 数据恢复底层能力国内最强（涉及坏盘、固件级修复，PC3000 国产替代方向）；② 服务 120+ 国家 5000+ 机构（包括 NASA、剑桥大学）—— 国内取证厂商中**国际化最深**；③ 国产 OS 数据恢复行业首发（统信 UOS / 麒麟原生数据恢复）。
- **短板**：① PC 取证分析能力（artifact 解析、报告生成）跟美亚 / 盘古石有差距，强项在硬件层；② 营收未上市披露；③ 移动取证占比较大、纯 PC 模块产品线相对单薄。
- **客户类型**：公安、军队、司法、金融行业 IT 资产数据恢复；央企存储介质数据找回。
- **价格**：取证一体机 15-50 万元；数据恢复服务按案例计费。
- **国外对标**：DeepSpar / Atola Insight（数据恢复硬件） + 部分 Tableau 功能。
- **CNVD 历史**：未检索到公开漏洞条目。

### 3.4 弘连网络（forensix.cn）

- **产品矩阵**：[上海弘连网络](https://www.forensix.cn/)，专精电子数据取证 + AI 分析，70+ 专利。核心产品：[寻迹多维数据分析专用机 XJ920](https://www.forensix.cn/xj920/)（账单/话单/聊天/卡口数据线索挖掘）、[火眼手机分析专用机 MT510/MT510D](https://www.forensix.cn/products/info.aspx?itemid=1129&lcid=6)、现场取证系列、实验室产品系列。
- **能力亮点**：① "AI + 取证"双轮驱动（早于美亚 Qiko）；② 寻迹 XJ920 把账单 / 话单 / 社交关系做成结构化关系图谱、可视化能力强；③ 同时持有计算机司法鉴定许可证、可对外做鉴定服务（厂商 + 鉴定所一体化，是商业模式特色）。
- **短板**：① PC 端计算机取证模块依赖现场取证设备承载、独立的纯软件分析包不显著；② 公司体量比美亚小一个量级；③ 跟"火眼"撞名美亚柏科（弘连"火眼"主指手机分析，美亚"火眼"涵盖更广，**采购时注意区分**）。
- **客户类型**：全国公检法司及行政执法机关。
- **价格**：未公开；XJ920 / MT510 一体机预估 20-40 万元。
- **国外对标**：Cellebrite Pathfinder（关系挖掘） + Magnet AXIOM 的组合。
- **CNVD 历史**：未检索到公开漏洞条目。

### 3.5 龙信科技（longxintec.com）

- **产品矩阵**：[苏州龙信信息科技](https://www.longxintec.com/)，13 个城市设研发/支持中心、服务 10000+ 执法单位。核心产品：[LX-A315 计算机快速采集系统](https://www.longxintec.com/?about_198%2F=)、[LX-A501 Pro 实验室勘验物联网取证工作站](https://longxintec.com/?about_111%2F=)、综合一体化取证设备（手机+介质+仿真+人物画像）、涉网案件研判串并平台。
- **能力亮点**：① 公安部 2024 年警用取证设备框架协议采购项目多款入围（[官网公司新闻](https://www.longxintec.com/?news_27%2F351.html=)）；② 现场快速采集 + 实验室深度勘验两套独立设计、流程清晰；③ 物联网取证（路由器 / 摄像头 / 智能终端）模块布局较早。
- **短板**：① 品牌知名度落后美亚 / 盘古石；② 产品文档公开化程度低（官网信息密度有限）；③ AI 分析模块薄弱。
- **客户类型**：基层公安、市场监管、网信办执法。
- **价格**：未公开。
- **国外对标**：Cellebrite + Magnet 中端组合，但分析深度不及。
- **CNVD 历史**：未检索到公开漏洞条目。

### 3.6 四川金舟 / 金诺信息

- **产品矩阵**：公开资料**非常稀少**。四川金舟司法鉴定技术研究院主营司法鉴定服务（属于消费方），金诺信息属于关联实体提供产品工具，但缺乏官网公开产品页。可能存在的产品包括取证一体机 + 数据恢复设备 + 鉴定工作站。
- **能力亮点**：① 西南地区（四川 / 重庆 / 云贵）公安系统覆盖较好；② 司法鉴定资质齐全、可一站式做"采集 + 分析 + 出鉴定报告"。
- **短板**：① 公开材料极少、不在主流厂商竞争力榜单（如 [2021 取证企业排行](https://www.secrss.com/articles/35011)）的前列；② 产品迭代节奏不透明；③ 难评估实际产品能力。
- **客户类型**：四川/西南公安、检察、法院。
- **价格**：未公开。
- **建议**：**互联网公司企业自用基本不会接触到这家**，仅作为完整性列出。如有西南地区驻场需求可作候补。

### 3.7 齐力软件 SP-EFD

- **产品矩阵**：搜索结果中仅找到 [齐力软件科技（广州）有限公司](https://m.qizhidao.com/qiye/company/3543836180b19ed293dca2ee8fe6db1a.html) 一条工商信息（注意是广州不是深圳），无活跃官网、无公开产品文档、无近期新闻。SP-EFD（电子证据搜索固定与分析）这个产品名仅在历史资料中出现过、**当前活跃度存疑**。
- **建议**：**对互联网公司无采购参考价值**，可视为"曾经存在过的取证厂商"。如要做完整厂商谱系研究，可作脚注；做选型决策直接跳过。
- **CNVD 历史**：无。

### 3.8 安天（Antiy）

- **产品矩阵**：[安天](https://www.antiy.cn/) 核心是 AVL 反病毒引擎和探海 NDR 流量检测系统（[安天探海](https://antiy.cn/Security_Product/PTD.html)）。**没有独立的电子数据取证一体机或取证大师式分析软件产品**。AVL SDK 支持文件对象 / 内存对象 / 网络流多类型检测、可作为取证流程中的恶意代码识别引擎。
- **能力亮点**：① AVL 引擎检测速度 200 PE 文件/秒、精准率 >99%；② [2004 年提出反病毒引擎延伸到取证](https://www.zhihu.com/question/528736394)的方法学；③ 探海作为旁路 NDR 自带 PCAP 留存和样本下载，**是网络流量取证的可行选项**。
- **短板**：① **不是 PC 取证主营厂商**，把它列入是因为很多人会问"安天能做取证吗"—— 答案是能做内存恶意代码识别和流量级证据留存，但**不能替代取证大师 / SafeAnalyzer**；② 探海的取证能力是 NDR 副产物，不是 forensic-grade chain of custody 产物。
- **客户类型**：政企信息安全、关基防护。
- **国外对标**：CrowdStrike Falcon Forensics 边缘能力 + ESET 引擎，但封装方式完全不同。
- **CNVD 历史**：安天作为漏洞研究方有大量上报记录，**反向作为产品出现的取证组件漏洞未公开**。

### 3.9 国保金泰

- **产品矩阵**：[北京国保金泰](http://www.jtsec.com.cn/About.aspx?Id=1) 1997 年成立、主营**安全隔离与信息交换系统**、安全单向数据自动导入系统、视频安全交换系统等。承担过五期 863 计划、十五/十一五国家科技计划项目。
- **能力亮点**：① 部委级涉密项目经验深（中办、国办、外交部、最高法等部署）；② 单向导入产品可用于"取证介质单向导入分析网"的物理隔离场景。
- **短板**：① **没有独立的取证一体机或分析软件主流产品线**；② 列入本表是因为部委级项目集成中常出现，**互联网公司基本接触不到**。
- **客户类型**：部委、保密单位、军队。
- **建议**：**互联网公司企业自用无关**，仅作背景完整性列出。

## 4. 国内 vs 国外 gap 分析

### 4.1 国外领先点（5 个）

1. **底层格式深度** — X-Ways Forensics / EnCase 对 NTFS USN journal / $LogFile / VSS / APFS snapshot / ReFS allocation 的解析深度跟国内还有 1-2 年差距，国内对 EXT4 64bit / btrfs / ZFS 解析普遍弱。
2. **内存取证生态** — Volatility 3 + Rekall + 商业 AXIOM Cyber Memory 形成的插件生态、kernel pool tag 库、SSDT/IDT hook 检测，国内厂商基本依赖把 Volatility 集成进自己的 GUI。
3. **跨国法庭采信度** — EnCase 自 1998 年起在美国/加拿大/英国/澳洲法庭立判例，新加坡、香港、欧盟法庭主流。国内产品出境基本只能用作"内部调查工具"。
4. **AI 辅助 triage 成熟度** — Magnet AXIOM Cyber 的 Magnet.AI、Cellebrite Pathfinder 的图片分类 / 聊天意图识别已经商用 3-4 年，国内 Qiko 大模型一体机刚推出。
5. **跨平台/云原生取证** — AWS EBS snapshot 取证、Azure VM disk forensics、Google Workspace export、Slack/Teams 取证 API 适配，国外厂商紧跟云厂商节奏，国内对国内云（阿里/腾讯/华为云）的官方取证 API 支持反而比国外还弱。

### 4.2 国内反超点（5 个）

1. **信创原生兼容** — 美亚 / 盘古石 / 效率源都在跑统信 UOS / 麒麟 / 飞腾 / 龙芯原生版本，EnCase / FTK 完全不适配国产 OS、X-Ways 只能跑 Wine。涉密 / 信创采购**唯一选择是国产**。
2. **国产应用 artifact 解析** — 微信 / QQ / 钉钉 / 飞书 / 抖音 / 拼多多 / 美团 / 滴滴 等 500+ 国产 App 的 SQLite 数据库结构、加密方式、binary log，国内厂商每月更新一次解析包；EnCase / AXIOM 顶多支持微信国际版。
3. **公安数据生态打通** — 取证报告直接对接公安大情报 / 警综 / 卡口 / 旅馆 / 实名 API，国外产品根本接不上。
4. **公安渠道下沉 + 驻场服务** — 市/县级公安都能找到本地工程师驻场，国外厂商在中国只有 1-2 个分销代理。
5. **价格 / 国产替代政策红利** — 国密 SM 系列加密支持、信创清单内采购、专项资金扶持。`取证大师`专业版 3.5 万元 vs `EnCase Forensic` 一年订阅 4000+ 美元（约 2.8 万人民币）+ 不含培训和升级 — 总拥有成本比拼时国内有明显优势。

## 5. 国内取证厂商的特殊性

**公安客户占比极高**：美亚柏科电子取证业务公开口径**公安客户占 70%+**，其余为检察、法院、海关、纪委监委。互联网公司、央企内审是次要客户。这导致 ① 产品 roadmap 严重跟公安刑侦/经侦案件需求走（涉网案件、电信诈骗、毒品案件、知识产权侵权）；② 报告模板天然按"鉴定意见书"格式设计、不一定贴合企业 IR 流程；③ 采购需要走政府/事业单位采购流程、互联网公司直接采购需要再适配。

**销售许可证政策变化（2023-07）**：[公安部令 168 号](https://www.gov.cn/gongbao/2023/issue_10786/202310/content_6912655.html) 废止了 1997 年起实施的《计算机信息系统安全专用产品检测和销售许可证管理办法》。自 2023-07-01 起，**停止颁发《计算机信息系统安全专用产品销售许可证》**（[网信办公告](https://www.cac.gov.cn/2023-04/17/c_1683373663312410.htm)），改由公安部认证中心和 ISCCC 等机构按 GB 强制性国标做"认证检测"。**对采购方的影响**：以前看"销许证号"判断合规性的做法失效，现在要看 GB/T 25069 系列国标的认证证书 + 等保 2.0 测评报告。[安全内参对行业影响的分析](https://www.secrss.com/articles/53880) 指出这是认证检测互认的关键一步。

**司法鉴定资质**：与产品销售并行的是"电子数据司法鉴定机构许可证"（省级司法厅颁发），美亚 / 弘连 / 效率源 / 龙信都设有关联鉴定所或与之深度合作。互联网公司采购 PC 取证产品时**不需要司法鉴定资质**，但走民事/刑事司法程序时仍需关联有资质的鉴定所做最终鉴定意见出具。

**HVV 实战驱动**：近年护网行动（HVV）红蓝对抗中应急响应取证需求激增，国内厂商在"快速镜像 + 内存采集 + IoC 检索 + 报告生成"链路的产品迭代明显加速。但 HVV 场景的产品（侧重快速 triage）跟传统刑侦取证（侧重链路完整、法庭采信）的产品要求有差异，企业自用更适合 HVV 化的产品（如盘古石现场取证、龙信 LX-A315 快速采集）。

## 6. 总结建议：互联网公司企业自用选型 Top 3

**首选场景：企业内部数据泄露调查 + HVV 应急响应取证**

1. **奇安信盘古石（SafeAnalyzer + SafeImager + 取证战星组合）** — 优点：① 跟奇安信整体安全栈（NDR / EDR / SOC）协同，互联网公司容易已经有奇安信生态；② 漏洞思维背景对 unlock / 绕过场景实用；③ 销售对接互联网客户的意愿和经验最强（相比美亚的公安导向）。劣势：底层 artifact 库厚度逊于美亚。**推荐场景**：HVV / IR / 内部审计 / 数据泄露调查。

2. **美亚柏科取证大师 + DC 系列硬件** — 优点：① 国产 App 解析最全（微信群聊 / 钉钉 / 飞书数据库直读），互联网公司内部沟通强依赖这些 App；② 信创兼容最成熟，**有等保 2.0 三级 / 信创清单要求时几乎是唯一可选**；③ 培训生态完善（美亚杯 + 教学体系）；④ 商务渠道下沉到地级市。劣势：销售流程偏公安/政企、对互联网客户响应不如盘古石灵活。**推荐场景**：信创合规要求 / 长期资产建设 / 法务诉讼支持。

3. **效率源 + 国外旗舰（EnCase / Magnet AXIOM）双栈** — 适合**头部互联网公司**（蚂蚁、字节、阿里、腾讯、美团级）：用效率源做数据恢复 + 国产 App 解析 + 信创合规层，用 EnCase / AXIOM 做底层 artifact 深度分析 + 跨境调查支持。预算允许时**国内 + 国外双栈是最佳实践**，因为 ① 跨境案件需要 EnCase 报告；② 国内合规需要国产产品；③ 互相对账可以发现单一工具的解析 bug。

**避免**：齐力软件（活跃度存疑）、四川金舟（公开材料不足）、安天 / 国保金泰当成 PC 取证主力（不是它们的主业）。**单独采购手机取证**时另起一个表（Cellebrite UFED / 美亚火眼 / 弘连 MT510），本文不展开。
